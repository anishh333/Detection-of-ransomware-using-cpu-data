System Architecture

Data Collection

Collects 5 HPC events (LLC-stores, L1-icache-load-misses, branch-load-misses, node-load-misses, instructions) and 8 disk I/O events at 5ms and 20ms intervals.

Feature Engineering

Computes mean, median, standard deviation, and inter-quartile range (IQR) for each 200ms data instance window containing 40 HPC and 10 I/O samples.

ML Classification

Tests 7 classifiers (RF, SVM, DT, kNN, XGBoost, DNN, LSTM). Random Forest with Integrated HPC+I/O model yields the best balanced accuracy of 0.92+.

Fast Detection

Detects active ransomware within 400ms with 98% probability. Resilient to varying user workloads when trained with multiple workload scenarios.

Based on: "Detection of Ransomware Attacks Using Processor and Disk Usage Data" — K. Thummapudi, P. Lama, R.V. Boppana (IEEE Access, 2023)
